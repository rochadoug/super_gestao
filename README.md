
# App Super Gestão

É um sistema simples de gerenciamento empresarial feito utilizando o Laravel como estrutura backend.

## Razão

Desenvolvido para fins de estudo do framework Laravel, o sistema tem a página inicial onde apresenta a empresa, e um formulário para contatos,
tem uma página "Sobre" que contém detalhes sobre a empresa, missão, visão e valor, etc. E uma página específica para contato também além do 
formúlario da página home.

-  No momento ainda está em desenvolvimento, em breve atualizo aqui com as demais features e melhores informações.
